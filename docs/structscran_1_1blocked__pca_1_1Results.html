template<typename EigenMatrix_, typename EigenVector_>

◆ components

template<typename EigenMatrix_ , typename EigenVector_ >
EigenMatrix_ scran::blocked_pca::Results< EigenMatrix_, EigenVector_ >::components

Matrix of principal components. By default, each row corresponds to a PC while each column corresponds to a cell in the input matrix. If Options::transpose = false, rows are cells instead. The number of PCs is determined by the rank used in compute().
